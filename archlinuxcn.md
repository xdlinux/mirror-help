# Archlinuxcn

## 介绍

Archlinux 中文社区仓库

## 收录版本

* rolling

## 收录架构

* x86_64

## 使用说明

编辑`/etc/pacman.conf`， 在文件末尾添加以下内容

```
[archlinuxcn]
Server = https://mirrors.xdlinux.info/archlinuxcn/$arch
```

然后请安装 archlinuxcn-keyring 包以导入 GPG key。
